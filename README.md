# Полезные промпты для AI-сервисов

## Фреймворки для промптов

| Фреймворк | Тип              | Расшифровка                                        | Назначение                                      | Описание                                                                         | Пример                                                                                                                                                    |
|-----------|------------------|----------------------------------------------------|-------------------------------------------------|----------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| CRISPE    | Системный        | Context, Role, Intent, Role, Parameters, Examples  | Универсальный каркас системного промпта         | Задаёт контекст, роль, цели, шаги, параметры и примеры; калибрует стиль и формат | Контекст: помощь Python dev; Роль: техредактор; Цель: исправлять ошибки; Шаги: уточнить, правка, код; Параметры: Py3.12, ≤150 слов; Примеры: вопрос→ответ |
| RISEN     | Системный        | Role, Input, Roles, Expectations, Nuances          | Пошаговое выполнение с критериями качества      | Фокус на последовательности, контроле качества и краевых случаях                 | Роль: аналитик; Вход: CSV sales,date; Шаги: пропуски, статистика, тренды, вывод; Ожидания: точность; Нюансы: сезонность, выбросы                          |
| RODES     | Системный        | Role, Objectives, Data, Execution, Safeguards      | Задачи с ограничениями и рисками                | Полезен для аудитов, политик и безопасности                                      | Роль: ИБ‑помощник; Цели: риски S3; Данные: бакеты, политики; Исполнение: шаги аудита; Безопасность: без чувств. данных                                    |
| ROLE      | Системный        | Role, Objectives, Limitations, Expectations        | Минималистичное задание поведения               | Быстро задаёт стиль ответа и критерии                                            | Роль: редактор резюме; Цели: читаемость; Ограничения: ≤120 слов, без воды; Ожидания: структура, активные глаголы                                          |
| CARE      | Системный        | Context, Action, Result, Estimate                  | Каркас с обратной связью и проверкой результата | Строит цикл «сделал — проверил — улучшил»                                        | Контекст: SQL отчёт; Действие: запрос; Результат: SQL+пояснение; Оценка: соответствие ТЗ, риски                                                           |
| APE       | Пользовательский | Action, Parameters, Examples                       | Короткие задачи с образцами                     | Задаёт что сделать, настройки и ожидаемый вывод                                  | Действие: перефразируй; Параметры: деловой тон, ~80 слов; Пример: исходник→стиль                                                                          |
| RACE      | Пользовательский | Request, Additional context, Constraints, Examples | Контекст и ограничения в запросе                | Избегает двусмысленности, закрепляет формат вывода                               | Запрос: чек‑лист лендинга; Контекст: SaaS B2B; Ограничения: ≤15 пунктов; Примеры: нумерованный список                                                     |
| PAR       | Пользовательский | Purpose, Audience, Requirements                    | Ориентация на цель и аудиторию                  | Привязывает ответ к KPI и уровню читателя                                        | Цель: вступление отчёта; Аудитория: топ‑менеджеры; Требования: 120–150 слов, 3 вывода                                                                     |
| AIDA      | Пользовательский | Attention, Interest, Desire, Action                | Убедительные тексты и промо                     | Структурирует копирайтинг для конверсии                                          | Тема: новая функция; Внимание: заголовок; Интерес: боль+инсайт; Желание: выгоды; Действие: CTA                                                            |
| STAR      | Пользовательский | Situation, Task, Actions, Result                   | Описание кейсов и опыта                         | Чёткое изложение процесса и эффекта                                              | Ситуация: падение конверсии; Задача: восстановить; Действия: A/B, фиксы; Результат: +12% за 10 дней                                                       |

## Генерация аудио

### Создание песни в стиле исполнителя
```
Напиши описание стиля песен <имя исполнителя> на английском для Suno 5. Ответом должно быть только перечисление существительных через запятую, например  heavy drums, female vocal… Общая длина не более 2000 символов.
```

### Создание разметки для стихов
```
Напиши пример структуры <имя исполнителя> на английском для Suno 5. Примером должен текст песни с указанием метаинформации.
```
